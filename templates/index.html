About

GlycoTorch is a python based web application for analysing Glycosaminoglycans (GAGs) and their complexes.

Users can upload GAGs structures and prepare input for GlycoTorch Vina, a GAG focused docking program. Download GlycoTorch Vina (Windows, Linux) or stay up to date with the source code on their Github repositories: GlycoTorch and GlycoTorch Vina.

Tools

GAG ligand input file preparation

Please select a .pdb file containing a GAG ligand to prepare for docking.
Structural information (i.e. φ and ψ glycosidic torsions and Cremer-Pople parameters describing ring conformations) will be displayed here.
 
 
 

Protein input file preparation

Please select a .pdb file of a protein (with hydrogens) to prepare for docking.
A docking "config" file, containing the search space and other parameters for docking, can be generated here.
