Docking with GlycoTorch Vina

1.0 Overview

This tutorial will cover preparing input files for GlycoTorch, as well as running and analysing docking results.

Users will need to download UCSF Chimera (or equivalent software) to add hydrogen atoms to a protein crystal structure. ( download here)

After such software is installed, the general outline of the tutorial is as follows:

  1. Add hydrogen atoms to the protein and extract co-crystallised ligand (e.g. in UCSF chimera).
  2. Prepare the GAG ligand for docking (on the GlycoTorch website).
  3. Prepare the protein for docking and select a binding site (on the GlycoTorch website).
  4. Run docking (on your local machine).
  5. View the results (e.g. in UCSF chimera).
